#Carousel Application

__To download the Carousel Application__
  
  Note: Use the ruby stack console.
  
  ---
	
	$ md workspace
	
	$ cd workspace
	
	$ git clone "link of app" 
	
	$ cd app
	
	$ rake db:migrate
	
	$ bundle install
	
	$ rails server
	
	After you run the server you can open the site by typing localhost:3000 in your browser.
